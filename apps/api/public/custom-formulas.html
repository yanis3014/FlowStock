Note sur IF()

Attention : La fonction IF évalue les deux branches (vrai et faux) avant de choisir. IF(CONSO > 0, STOCK / CONSO, 0) provoquera une erreur si CONSO = 0. Utilisez plutôt : STOCK / MAX(CONSO, 0.001) pour éviter les divisions par zéro.
