# Story 9.3: Refonte du Dashboard Actionnable

**Status:** Ready

## Story

**As a** gérant de PME,  
**I want** un dashboard moderne avec chargement automatique et boutons d'action intégrés,  
**so that** je vois immédiatement l'état de mes stocks et je peux agir sans clic superflu.

## Acceptance Criteria

**Given** un utilisateur authentifié  
**When** j'accède au dashboard  
**Then** des Skeletons avec effet shimmer s'affichent pendant le chargement des données  
**And** les données sont chargées automatiquement au montage (useEffect / Server Components selon architecture)  
**And** aucune action manuelle type « Charger les estimations » ou « Rafraîchir » n'est requise pour voir les infos  
**And** chaque alerte de stock critique propose un bouton d'action stylisé (ex. « Commander ») avec états loading et confirmé  
**And** les cartes de données ont une hiérarchie visuelle claire (ombres, contrastes, iconographie)  
**And** le design respecte les standards SaaS professionnels (Shadcn Card, Badge, Button)

## Tasks / Subtasks

- [ ] Task 1 — Chargement automatique des données
  - Appeler les API dashboard au montage (useEffect ou Server Components)
  - Supprimer les boutons « Charger les estimations » / « Rafraîchir »
  - Gérer les états loading, error et success

- [ ] Task 2 — Skeletons avec effet shimmer
  - Créer des Skeletons qui imitent la forme des cartes et des alertes
  - Appliquer un effet shimmer pendant le chargement
  - Réutiliser le pattern défini dans docs/front-end-spec.md §6 (Skeleton)

- [ ] Task 3 — Cartes de données professionnelles
  - Utiliser Shadcn Card pour les blocs de statistiques
  - Appliquer ombres, contrastes et iconographie cohérente
  - Respecter la hiérarchie visuelle (primary, success, warning, error)

- [ ] Task 4 — Boutons d'action par alerte
  - Chaque alerte de stock critique affiche un bouton « Commander » (ou équivalent)
  - États : default, loading (spinner intégré), confirmé (check vert)
  - Intégrer le flux de génération de bon de commande (docs/front-end-spec.md §4.1)
